# project-builder-lecture

로또 번호 생성기 🎱 — Git / GitHub 학습용 프로젝트입니다.

## 소개

버튼을 누르면 1~45 중 중복 없이 6개의 로또 번호를 뽑아주는
간단한 웹 애플리케이션입니다. 번호대별로 실제 로또처럼 공 색깔이 다릅니다.

## 데모 (GitHub Pages)

👉 https://parengjin3.github.io/project-builder-lecture/

## 구성

- `index.html` — 로또 번호 생성기 (HTML + CSS + JavaScript)
- `.gitignore` — Git이 무시할 파일 목록
- `README.md` — 프로젝트 설명 문서

## 실행 방법

- `index.html` 파일을 더블 클릭해서 브라우저로 열거나,
- 위의 GitHub Pages 데모 링크로 접속하면 됩니다.
